# 📊 Sales Analytics Dashboard

A beginner-friendly interactive Sales Analytics Dashboard built using Python and Streamlit.

## Features

- 📈 KPI Dashboard
- 📊 Interactive Charts
- 🔍 Sidebar Filters
- 🤖 AI Business Insights
- 📥 CSV Download
- 📊 Excel Download
- 📄 PDF Report Download

## Technologies Used

- Python
- Streamlit
- Pandas
- Plotly
- OpenPyXL
- ReportLab

## Project Structure

```
Sales-Dashboard/
│
├── app.py
├── data/
├── assets/
├── requirements.txt
└── README.md
```

## Run Locally

```bash
pip install -r requirements.txt
streamlit run app.py
```
